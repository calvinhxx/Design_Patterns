## 提交规范 

### [1.blockly-commit-message-guide](https://developers.google.com/blockly/guides/contribute/get-started)

**Each commit should have the format:**
```
<type> : <description>
[optional body]
[optional footer(s)]
```

- Type
  * chore   `构建工具或者依赖的变动`
  * deprecate   `弃用功能`
  * feat    `新功能`
  * fix     `修复BUG`
  * release `版本发布`
  * docs    `文档改动`
  * style   `代码样式改动`
  * refactor `重构`
  * test    `增添test`

**Breaking changes**
`Commits that make breaking changes should append a ! after the type of the commit.`
